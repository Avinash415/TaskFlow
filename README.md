# TaskFlow

TaskFlow is a modern Enterprise Task Management System built to manage projects, tasks, categories, and users with secure authentication and role-based access control.

## 🚀 Features

- JWT Authentication
- Role-Based Access Control
- User Management
- Project Management
- Task Management
- Category Management
- Responsive Dashboard

## 🛠 Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- PostgreSQL

### Frontend
- React.js (Vite)
- Tailwind CSS
- React Router
- Axios

## 📦 Installation

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
